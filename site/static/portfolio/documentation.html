The first of 3 posts I wrote for the excellent Docs Like Code blog. This one explained how we were able to document multiple versions of our product from a single source of truth using the Go language inside our static site generator.

"Docs.balsamiq.com: Our New Static Documentation Site, Powered by Hugo"

A post on the Balsamiq blog that explains the redesign and transition of our documentation and support sites from an old CMS to a modern static site.

"Behind the Scenes of Our Documentation Process"

A Balsamiq blog post about how we triaged support issues and turned frequent questions and issues into relevant help articles and documentation.
