Appendix C - Product Name Images

A product name image contains text that describes the name of an application. Using images for displaying application product name text provides the ability to present names using a Sun-branded font face, and does not create translation issues since product names are not translated.

Application development teams need to provision product name images for their applications. A utility exists to enable teams to create product name images, it is referred to as the “pngomatic”. The pngomatic generates product name image files for use within the primary masthead, secondary masthead, version page and login page.

The pngomatic creates a product name image from a product name text string, input into the utility. The utility outputs the text string as a .png, product name image file.

Font styling of the product name image text is handled by the pngomatic utility. Capitalization and some specific characters, for trademarks and two-part names, need to be included in the text string input to achieve a correct image output.

Use headline capitalization for product name image text. See 2.5 Text Capitalization.

Localization is not required for product name images, although Alt Text must be provided. The Alt Text should contain the name of the product as it appears in the product name image.

Include trademark symbols within a product name when the name is trademarked, as required by the Trademarks Group at Trademarks@sun.com.

When presenting a product suite, use a two-part name. The first part of the name should be the application suite name followed by the specific product name, which is the second part. For example, Sun StorEdgeTM SAM-QFS Manager. A two-part name is output by the pngomatic utility with unique formatting depending on the presentation context.

For more information and detailed instructions in using the pngomatic to create product name images, see “Creating Product Name Images for your Woodstock Masthead” at: http://bigblock.east.sun.com:8080/WoodstockWiki/Wiki.jsp?page=PngOMatic.

Questions about the pngomatic utility should be directed to: woodstock-dev@sun.com.
