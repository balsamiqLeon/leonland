Navigation to Specific Instance Among From 25 to 150 Objects

1. Common Tasks Page
   
User clicks on a common task button.
2. Pop-up dialog
 

The user can search, which narrows down the list, or directly browse.

Clicking the Cancel button closes the pop-up and returns the user to the Common Tasks page.
User selects one of the objects and clicks the OK button.
3. Selected Object's Page
  The pop-up window closes and the user is taken to the instance of the object selected.
