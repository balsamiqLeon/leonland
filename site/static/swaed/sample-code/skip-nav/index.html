Overview

This page includes sample HTML for the skip navigation tag. This tag is used to achieve section 508 compliance for 1194.22, Paragraph (o). For more information see 1.4 Resources for Application Designers. A Sun technical note on this section states:

"Pages that provide five or more navigation links in the header or left side bar of the page must provide a means of skipping over these links. This can be done by placing the navigation regions in their own frames or by providing a skip link method similar to the one used on Sun's home page. (You will need to view the page source to see the skip links.)"

This requirement applies to the masthead, tabs, and potentially to breadcrumbs (though this is debatable).

The method used here is the same as the one used on the sun.com page, It uses a hyperlinked single-pixel transparent gif (not visible in the browser page) with an alt tag that, when read by an accessible technology such as a screen reader, informs users that the link will allow them to skip over the navigation section. The hyperlink links to an anchor that is placed after the stuff to be skipped. The tag should set down both the hyperlinked transparent gif and the ending anchor, and should allow application developers to specify the anchor name, the alt text to be placed on the image, and the location of the ending anchor.

This tag should come in two forms, one where the hyperlink is specified inside of an element, such as a table, and one that needs to stand alone on the page between elements. In the latter case, the <a href> is specified inside of a <div> and in the former case it is on its own. When the <a href> is specified inside of a <div> it may need to have a class specified to ensure that the resulting single pixel that will span the entire width of the window when this tag is called will have the appropriate background color.
