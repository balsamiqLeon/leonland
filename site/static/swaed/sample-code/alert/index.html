Overview

The inline alert tag covers the five types of messages that appear at the tops of content pages: Error, Warning, Information, Question and Success. These messages each have a particular icon, a header, which appears in boldface font and additional information, which appears in regular weight font. The guidelines for inline alerts can be found in 14.1 Inline Alerts.

Messages can optionally have a link associated with them. When links are present, they should appear either to the right of the message text or beneath and left aligned with the message text. The following image: should be included in the href preceding the link text. The image itself should have an alt attribute equal to the title of the link and if the link will open a new window, the link should specify that a new window is being opened. If the link does not open a new window the title should contain some descriptive text.
