Overview

This is sample HTML for inline help text that appears embedded in page content. The page corresponds to 11.1 Inline Help. The help text is of two varieties: Page inline help and Field inline help. Page inline help text appears at the top of a section or content area and is intended to provide help on that entire section or page. Field inline help text pertains to and appears directly beneath a particular field or group of closely related fields.

Both types of help text may include hyperlinks inside the text itself and/or at the end of the text. When the links appear at the end of the text, the following image: should be included in the href preceding the link text. The image itself should have

alt="" and the title for the link should specify that a new window is being opened, e.g., "Help on Projects (Opens New Window) ".
