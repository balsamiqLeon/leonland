Release Notes

One noteworthy difference between this component and most others is that it often occupies the entire page. Because of this certain styles need to be applied to the BODY tag.

This component was styled using descendant selectors for future versioning and theme flexibility. Note that the class of the outermost DIV bears the name and version number of the component.
