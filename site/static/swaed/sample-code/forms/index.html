Overview

This page corresponds to 7.2 Elements Used for Choosing Data and 7.3 Text Elements and includes sample HTML and styles for the following tags:

  • Selectable List Tag
  • Radio Button Tag
  • Drop Down Menu Tag
  • Text Area, Text Field, Password Tags
  • Checkbox Tag

All of these tags need styles assigned to them, primarily in order to control their appearance when they are disabled. All of these elements have to be dynamically enabled and disabled. When they are disabled, not only do the tags need to have disabled="disabled" added as an attribute, but their style must also be changed to its disabled version (e.g., TxtFld => TxtFldDis for text fields). See the examples below for details

The form elements will almost always need to be used in conjunction with the label tag.

Release Notes

IE6 on Windows does not support disabling the individual <option>'s within a <select>. It simply ignores the specification to disable it. Same goes for NS6.2.2, although NS7 and recent Mozilla builds support disabling menu items.
