Overview

This is the example HTML for navigation trees, which are discussed in 5.4 Tree Details.

Tooltips are specified on alarmed objects in compliance with the guidelines specified in 11.5.7 Tool Tips for Icons with Alarm Badges. The basic form is [Alarm: <level> ] <object type>: <object name>, where level can be Down, Critical, Major, or Minor. For example: "[Alarm:Critical] Server: Crocker". This is specified via the alt and title attributes in the <img> tag and the title attribute of any <a href's> associated with that object (such as on the gif and on the link text itself).

Alts and titles are set in the same way on non-alarmed object graphics in the tree using the form <object type>: <object name>. For example: "Server: Crocker".
