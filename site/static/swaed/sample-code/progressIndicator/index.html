Release Notes

The examples for the progress indicator are rendered on the client side. As such, a simple "View->Page Source" will not reveal its markup. You will need to use Mozilla FireFox and the extension, "View Source Chart" or "Web Developer".

"View Source Chart" will add another option to the View menu in your Mozilla FireFox browser. Then you can navigate to View->View Source Chart to reveal a neatly formatted chart of the HTML and JavaScript source for that example.

"Web Developer" adds a host of other functions useful to any Web Designer/Developer including one for viewing the generated source (generated by the client). Once installed, you can navigate to Tools->Web Developer->View Source->View Generated Source. It won't be as neatly formatted as the "View Source Chart" extension, but if you are going to copy and paste it to work on it locally, then that may be less important to you.

This component was styled using descendant selectors for future versioning and theme flexibility. Note that the class of the outermost DIV bears the name and version number of the component.
