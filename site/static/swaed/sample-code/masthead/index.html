Overview

This tag implements the masthead for main windows and pop-up windows and corresponds to Chapter 4, Mastheads. The main window masthead has 5 sections, four of which are required. The size of the corporate logo sections is fixed in terms of height and width.

The optional status area, when used, can contain information such as alarms, time stamp information, and progress indication. Included in the Examples below is HTML that allows those using Assistive Technologies (AT) to skip over the entire masthead, including all images and links (see the skip navigation sample HTML page for more information on the method used here). If the optional Status Area is used, users will first be presented with the option to skip over the entire masthead and also to skip the the status area of the masthead. If the optional status area is not present, then the user is only given the option of skipping over the entire masthead. The alt text for the link allowing AT users to jump over the entire masthead is: "Jump to End of Masthead Section" and when the option is provided for users to jump to the status area, it is:"Jump to Status Area of Masthead Section".

The pop-up window masthead only has the product name section. The product area and table occupy 100% width of the page. There is no skip navigation HTML included in pop-up window mastheads.
