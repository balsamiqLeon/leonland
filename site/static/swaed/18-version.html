18 Version Page

The Version Page is provided for each application using the Sun Common UI Component Library (in open source, the Project Woodstock components). The basic Version Page design can be customized for an individual application by using application-specific data (product name, version number, and a legal notice) which is provided by an application development team.

The Version Page may be accessed by using the Version button, located in the following two places:

  • Within the Utility bar in the upper left side of a Login page. For more information see: 17.1.1 Utility Bar.
  • Within the Utility bar in a primary window masthead. The Version button is presented next to the Applications button. For more information, see 4.1.1 Utility Bar.

18.1 Version Page Elements

The Version Page is comprised of the following elements:

  • Branding image
  • Product name image
  • Java logo
  • Version number of the application
  • Legal notice or proprietary rights notice (PRN)
  • Close button: Focus begins on the Close button, allowing the window to be closed by pressing Enter.

18.1.1 Branding Image with Sun Logo

The branding image is an element within an overall design system which delivers a consistent brand presentation. The branding image includes the Sun logo, which functions as an endorsement of the Java brand. This brand element is reserved for use in web applications developed by Sun Microsystems. Otherwise, this element should be replaced with a brand element appropriate to the company or product that they represent.

To ensure a consistent brand presentation, use the Version Page design provided through the Sun Common UI Component Library (in open source, the Project Woodstock components).

18.1.2 Product Name Image

The Version Page is always titled with the name of the product. The product name is embedded in a .png image, and is presented to the right of the branding image at the top of the screen. An application development team will need to provision the product name image for their application. For more information on creating product name images, see Appendix C, Product Name Images.

18.1.3 Java Logo

The presence of the Java logo reinforces the role of Java as the lead brand. Application teams should contact their brand representative if they are uncertain about which brand marks are appropriate for their application. This brand element is reserved for use in web applications developed by Sun Microsystems. Otherwise, this element should be replaced with a brand element appropriate to the company or product that they represent.

For additional information on branding guidelines, see 1.4 Resources for Application Designers.

18.1.4 Application Version Number

Each revision of an application has a different version number. This location on the Version Page might be the only place that the application version number appears, other than on actual product packaging. This information is important when technical support personnel are troubleshooting application problems.

18.1.5 Legal Notice

The legal notice or proprietary rights notice (PRN) is a block of text that appears on the Version Page and is specific to the application associated with it. It appears on a gray background.

To protect Sun's intellectual property rights (for example, copyrights and trademarks), Sun must give adequate notice of those proprietary rights on all its products. Additionally, Sun is sometimes required by contract to attribute the intellectual property rights of third parties. The same will be true for other companies.

For additional information on developing a custom legal notice, see 1.4 Resources for Application Designers.

A custom legal notice must be composed and approved by a representative from the application development team's legal division. Every application requires this legal notice to protect Sun and its partners' intellectual property.

 For products releasing worldwide that will not be localized, include both the English and the French versions of the legal notice. If a French version of the product is available, do not include a French legal notice in the English version of the product.
