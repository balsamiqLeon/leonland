5.5.1 Access to the Common Tasks Page

The following navigational styles could be used to navigate to the Common Tasks page. At least one of these access methods should be available at all times so the user can return easily to the common tasks page.

R1 The button/link, tree control, or tab for getting back to the Common Tasks page should always be visible and available.

  • Tree
    The tree appears in a left pane with common tasks above the root node. The common tasks themselves do not appear in the tree; they are shown to the right of the tree on the Common Tasks page.

  • Tabs
    This is the standard tab set where the tabs are present on all primary pages. The Common Tasks tab should be the left-most, top-most tab as shown below.

  • Drill-down Through Breadcrumbs
    If the user drills down below the top set of tabs, the Common Tasks tab is presented in the breadcrumb bar as shown below.

If a new set of tabs occurs at a lower level, the breadcrumbs are shown the same way above those tabs but on a gray background.
