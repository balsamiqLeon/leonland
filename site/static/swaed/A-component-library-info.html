Appendices > Appendix A

Appendix A - Component Library Information

If you are developing an application, you should use the Sun Common UI Component Library (in open source, the Project Woodstock components), which will produce guidelines-compliant HTML. The components in this library have been implemented for the elements covered in these guidelines.


For a complete list of these components, see the Full Component List at:
https://woodstock.dev.java.net/Preview.htm

To download the Component Library, see the Woodstock Download page at:
https://woodstock.dev.java.net/Download.htm

For additional information regarding this release of the Sun Common UI Component Library, see the Project Woodstock page at:
https://woodstock.dev.java.net/index.html

Note: To create a Sun branded application, see the instructions for obtaining Sun branded images and creating a Sun Branded application at: http://bigblock.east.sun.com:8080/WoodstockWiki/Edit.jsp?page=SunBrandedApps
